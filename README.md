

# 🧠 NIRNAY

🚀 **Multi-Agent AI Platform for Automated R&D Proposal Evaluation**

AutoGen • LangChain • SBERT • GCN • PPO-RL • Neo4j • Milvus • Flask

---

## 📌 Overview

NIRNAY is an **intelligent multi-agent evaluation platform** designed to automate and optimize the assessment of **R&D proposals**.

It addresses critical challenges like **manual bias**, **subjective evaluation**, and **inefficient review processes** by leveraging **AI-driven decision systems, knowledge graphs, and reinforcement learning**.

The system enables **accurate, scalable, and explainable evaluation** of proposals across multiple dimensions including **novelty, plagiarism, and financial feasibility**.

---

## ⚡ Key Features

### 🤖 Multi-Agent Orchestration

* Built using **AutoGen + LangChain**
* Specialized agents for:

  * 📄 Plagiarism Detection
  * 🧪 Novelty Assessment
  * 💰 Financial Validation

---

### 🧠 Advanced ML & NLP Pipeline

* **SBERT embeddings** for semantic similarity
* Custom **NER models** for domain-specific extraction
* **Graph Convolutional Networks (GCN)** for novelty scoring
* **PPO Reinforcement Learning** for continuous optimization

---

### 🕸️ Knowledge Graph Intelligence

* **Neo4j-based knowledge graph** for contextual reasoning
* Captures relationships between research domains, ideas, and prior work

---

### 🔎 Vector Search & Retrieval

* **Milvus vector database** for high-speed similarity search
* Efficient retrieval of related research and prior proposals

---

### 📊 Explainable AI Dashboard

* Transparent scoring and reasoning outputs
* Improves trust and interpretability in decision-making

---
